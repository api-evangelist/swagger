# Swagger

Swagger is an open-source framework by SmartBear for designing, building, documenting, and consuming RESTful APIs using the OpenAPI Specification. Originally created by Wordnik in 2011 and renamed OpenAPI in 2016, the Swagger toolset includes Swagger UI (interactive docs), Swagger Editor (spec editor), and Swagger Codegen (code generation). The current OpenAPI Specification is version 3.2.0 (September 2025).

**URL:** [https://swagger.io](https://swagger.io)

## APIs

### Swagger UI

Renders OpenAPI specifications as interactive API documentation with try-it-out functionality.

**Human URL:** [https://swagger.io/tools/swagger-ui/](https://swagger.io/tools/swagger-ui/)

### Swagger Editor

Browser-based editor for writing and validating OpenAPI and AsyncAPI specifications.

**Human URL:** [https://swagger.io/tools/swagger-editor/](https://swagger.io/tools/swagger-editor/)

### Swagger Codegen

Generates server stubs, client SDKs, and documentation from OpenAPI specs in 40+ languages.

**Human URL:** [https://swagger.io/tools/swagger-codegen/](https://swagger.io/tools/swagger-codegen/)

### OpenAPI Specification

The language-agnostic standard (OAS 3.1.1 / 3.2.0) for describing HTTP APIs, governed by the OpenAPI Initiative.

**Human URL:** [https://swagger.io/specification/](https://swagger.io/specification/)
